# Exercício de Recuperação

Um caixa eletrônico possui apenas notas de **R$ 100,00**.

O programa deverá permanecer em execução até que o usuário digite o valor **0**.

Enquanto o valor informado for diferente de **0**:

- Leia o valor do saque desejado;
- Caso o valor não seja múltiplo de **R$ 100,00**, exiba a mensagem:
  **"Não é possível realizar o saque."**
- Caso contrário, informe a quantidade de notas de **R$ 100,00** necessárias para realizar o saque.

Quando o usuário digitar **0**, o programa deverá ser encerrado.

## Exemplo

```text
Digite o valor do saque: 500
Quantidade de notas: 5

Digite o valor do saque: 350
Não é possível realizar o saque.

Digite o valor do saque: 1000
Quantidade de notas: 10

Digite o valor do saque: 0
Programa encerrado.
```

**Entregue o código-fonte devidamente comentado e funcionando corretamente.**
